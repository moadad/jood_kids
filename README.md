# JOOD KIDS Sales Pro

نسخة واجهات احترافية جاهزة للرفع على GitHub وتشغيلها كموقع / PWA.

## التشغيل
```bash
python -m http.server 8000
```
ثم افتح:
```text
http://localhost:8000
```

## الموجود في النسخة
- لوحة تحكم احترافية
- شاشة فواتير مع خصم بالقيمة أو النسبة
- طباعة A4
- حفظ كصورة PNG
- العملاء
- المواد
- توليد باركود السيري من باركود القطعة + عدد السيري
- تصدير Excel بصيغة XLS
- مستخدمون وصلاحيات عرضية داخل الواجهة
- PWA قابل للتثبيت
- إعدادات Firebase و Cloudinary ضمن المشروع
